#### ğŸ§ª Spray Guidance (NEW FEATURE)
Helps farmers calculate the **exact amount of pesticide or spray chemical** needed based on their field size.

**How it works:**
- Farmer enters:
  - Crop type  
  - Field size (acre/hectare or local unit)  
  - Medicine / pesticide name(s) they already have  
- System calculates:
  - Required **ml quantity of each medicine**  
  - Recommended **water quantity** for spraying  
- Results are shown in a **simple, easy-to-understand format**

This reduces guesswork, prevents chemical misuse, and supports safe farming practices.
